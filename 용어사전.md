# MyAsset 용어 사전

| 한국어      | 영어                           | 정의                                                     |
|----------|------------------------------|--------------------------------------------------------|
| 사용자      | user                         | MyAsset 서비스를 이용하는 개인 사용자로, 자신의 금융 자산과 소비 내역을 통합 관리한다.  |
| 자산       | Asset                        | 사용자가 보유한 은행계좌·현금·증권계좌 등으로 구성된다.                        |        
| 은행계좌     | bank(Account)                | 사용자 명의의 입출금 계좌로, 금융결제원 API 등을 통해 연동되어 잔액 및 거래내역을 조회한다. |
| 현금       | cash                         | 사용자 보유 현금을 의미하며, 시스템 내에서 사용자가 직접 입력·관리하는 자산이다.         |
| 증권계좌     | Securities(Account)          | 사용자가 보유한 증권 계좌로, *(연동내역 편집 필요)*                        |
| 거래내역     | transaction                  | 자산에서 발생한 수입 또는 지출 기록                                   |
| 예산       | budget                       | 사용자가 월 단위로 설정하는 지출 한도로, 미설정 시 과거 소비 패턴을 기반으로 자동 산정     |
| 카테고리     | category                     | 거래내역을 분류하기 위한 기준으로, 사용자가 생성·수정·삭제할 수 있다.               |
| 보유주식종목   | Holding                      |                                                        |
| 주식종목뉴스   | news                         | 사용자가 보유한 주식 종목과 관련하여 외부에서 크롤링된 금융 뉴스 정보이다.             |
| 알림       | alarm                        | 예산 초과, 비정상 지출, 투자 종목 급변 이벤트 발생 시 사용자에게 전달되는 알림이다.      |
| 자산 포트폴리오 | portfolio                    | 사용자가 보유한 모든 자산을 통합하여 자산 구성 비중과 변동 추이를 시각적으로 제공하는 정보    |
| 로그       | log                          |                                                        |
| 소셜로그인    | login                        | Kakao, Google, Naver를 통한 소셜로그인                         |
| 회원가입     | register / registerFromOAuth | 어플리케이션 계정생성 / 소셜회원가입                                   |
| 회원탈퇴     | withdraw                     |                                                        |
| 입금       | income                       |                                                        |
| 출금       | expanse                      |                                                        |
