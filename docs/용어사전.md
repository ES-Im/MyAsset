# MyAsset 용어 사전

| 한국어      | 영어                            | 정의                                                    |
|----------|-------------------------------|-------------------------------------------------------|
| 사용자      | user                          | MyAsset 서비스를 이용하는 개인 사용자로, 자신의 금융 자산과 소비 내역을 통합 관리한다. |
| 자산       | Asset                         | 사용자가 보유한 은행계좌·현금·증권계좌·카드 등으로 구성된다.                    |        
| 은행(계좌)   | bank(Account)                 | 사용자 명의의 입출금 계좌                                        |
| 현금       | cash                          | 사용자 보유 현금                                             |
| 증권(계좌)   | Stock(Account)                | 사용자가 보유한 증권(주식) 계좌                                    |
| 신용카드     | card                          | 사용자 명의의 신용카드                                          |
| 거래내역     | transaction(tran)             | 카드/현금/은행계좌 대상 일어나는 거래내역                               |
| 주식거래내역   | stock_trade                   | 증권계좌 대상 주식 거래내역                                       |
| 보유주식종목   | Holding_stock                 |                                                       |
| 예산       | budget                        | 사용자가 월 단위로 설정하는 지출 예산                                 |
| 카테고리     | category                      | 거래내역을 분류하기 위한 기준으로, 사용자가 생성·수정·삭제할 수 있다.              |
| 주식종목뉴스   | news                          | 사용자가 보유한 주식 종목과 관련하여 외부에서 크롤링된 금융 뉴스 정보이다.            |
| 알림       | alarm                         | 예산 초과, 비정상 지출, 투자 종목 급변 이벤트 발생 시 사용자에게 전달되는 알림이다.     |
| 자산 포트폴리오 | portfolio                     | 사용자가 보유한 모든 자산을 통합하여 자산 구성 비중과 변동 추이를 시각적으로 제공하는 정보   |
| 로그       | log                           |                                                       |
| 소셜로그인    | login                         | Kakao, Google, Naver를 통한 소셜로그인                        |
| 회원가입     | register / registerFromOAuth  | 어플리케이션 계정생성 / 소셜회원가입                                  |
| 회원탈퇴     | withdraw                      |                                                       |
| 유입       | inflow                        | 계좌 입금, 현금 유입 등                                        |
| 출금       | outflow                       | 계좌 출금, 현급 유출 등                                        |
| 고정지출     | fixed Expenses (FE)           | 주거비, 공과금, 통신/정보비, 보험/저축비, 교육/양육비, 교통비 등               |
| 변동지출     | Variable Expenses ( VC)       | 식비, 생활비, 의료비, 자기계발비, 여가비                              |
| 알림       | notification (noti)           | 서비스 알림                                                |
